What is Ave ?

Ave is a general purpose programming language that compiles to Javascript. It aims to keep only the crucial parts of your code and get rid of the uneccessary ones, enforcing a clean, readable codebase. 

						
  fn greet(name)
    console.log(`Hello world, this is ${name}!`)

  greet('Ave') #Hello world, this is Ave!

Expressive syntax

Ave adds a layer of syntactic sugar over vanilla Javascript, adding tonnes of new features to increase productivity and lessen the load on you, as a developer. The indented block structure is Python inspired to make your code more readable. 

						
  enum Week
    Monday, Tuesday, Wednesday,
    Thursday, Friday, Saturday,
    Sunday 
  console.log(Week.Monday) #Monday

Open Source


Ave is completely open source. It's source code lives right here on github. Feel free to take a look and contribute, suggest changes or report bugs.

In the end, its JS
